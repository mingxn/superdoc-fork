# Integration Examples

Examples showing how to integrate SuperDoc with various platforms and environments.

## Examples

| Example | Description |
|---------|-------------|
| [nextjs-ssr](./nextjs-ssr) | Next.js with server-side rendering |
| [nodejs](./nodejs) | Node.js backend for document processing |
| [chrome-extension](./chrome-extension) | Chrome browser extension |
| [word-addin](./word-addin) | Microsoft Word add-in |
| [slack-redlining](./slack-redlining) | Slack integration for document redlining |

## Running an Example

```bash
cd <example-folder>
npm install
npm run dev
```

See individual example READMEs for platform-specific setup instructions.
